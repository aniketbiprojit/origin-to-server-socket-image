# origin-to-server-socket-image

```sh
cd client
yarn
yarn start
```
In `client/src/index.js` change from `<AppModelOnly/>` to `<App>`

```sh
cd server
npm i
node index.js
```


Go to `http://localhost:3000`

Image processing on server 'frame by frame'.
